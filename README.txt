/***** Compilation ******/
In the root project directory, execute :
  1. qmake
  2. make

/***** Execution *****/

Execute "Othello" in a shell, or double-click the icon.
The game begin when the "start" button is pushed. You can restart at any time by pushing this button again.
You are then the first player, and can try to beat the computer.
Enjoy the game !
